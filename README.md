# Kasper Desert Survival RPG

RPG project created for my Java final.

## How to Run

Compile:

javac *.java

Run:

java Main

Or run the executable JAR:

java -jar KasperDesertSurvivalRPG.jar

## Game Theme

You awaken alone in the desert with little water, can you survive?

## Current Features

* Player class system
* Enemy system
* Desert story introduction
* Turn-based combat
* Attack option
* Search for water option
* Run option
* Sand Raider encounter
* Giant Sand Slug encounter
* Boss Scorpion encounter
* Victory screen
* Console-based RPG gameplay
* Scanner user input
* Object-oriented programming structure

## Character Class

* Wanderer

Class Buff: Desert Survivor

Special Ability: Enhanced Water Foraging

## Project Structure

* Main.java -> Runs the game loop and story progression
* Player.java -> Stores player data, stats, and actions
* Enemy.java -> Stores enemy data and combat behavior
