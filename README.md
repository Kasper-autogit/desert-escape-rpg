# Desert Escape RPG

RPG project created for my Java final.

## Game Theme
You awaken alone in the desert with little water and no memory of how you arrived there. The sun burns overhead as dangerous enemies roam the sands. Your goal is to survive and eventually find a way out of the desert.

## Current Features
- Player class system
- Enemy system
- Desert story introduction
- Interactive combat choices
- Attack option
- Heal option
- Run option
- Console-based RPG gameplay
- Scanner user input
- Object-oriented programming structure
